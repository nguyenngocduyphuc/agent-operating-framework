# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in the Agent Operating Framework (AOF), please report it responsibly.

**Do not open a public GitHub issue for security vulnerabilities.**

Instead, contact the maintainer directly or use any private communication channel available.

Please include:
- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We will acknowledge receipt within 48 hours and aim to provide a fix or mitigation plan within 7 days.

## Supported Versions

Current development is focused on the latest version. If you're using a forked version, please maintain it independently.

## Scope

This policy applies to the core AOF framework code and its official distributions.
